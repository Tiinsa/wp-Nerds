В над проектом работала ноябрь 2017 - январь 2018г.
В ходе проекта было выполнено:

1. Изучена установка ВП на локальную машину.
2. Изучены основные принципы построения тем, натяжка верстки на движок ВП.
3. Выполнена натяжка верстки сайта на движок ВП.
4. Изучена работа Customizer API ВП.
5. Реализована настройка карты Google.maps (изменение координат, зума, маркера) через админ панель (Customizer API).
	5.1 Изучены методы работы с объектами Google.maps,центрирование карты, анимирование маркера, перемещение маркера, получение координат центра карты.
6. Реализована настройка слайдов слайдера и раздела FEATURES (изменение картинок и надписей) через админ панель (Customizer API).
7. Реализована страница каталога с возможностью постраничного разделения, фильтрации и сортировки по нескольким параметрам.
   7.1 Изучен механизм сохранения и получения парметров фильтрации через сессии, GET запросы.
   7.2 Написан скрипт ползунка и выполнена привязка данных ползунка к данным инпутов.



По проекту реализованый функционал:
- Настройка местоположения офиса на карте через админ панель.
- Настройка содержимого слайдов (текст, изображение) слайдера в шапке на главной странице через админ панель.
- Настройка карточек Возможности (текст, изображение) через админ панель.
- Настройка содержимого главной страницы средствами ВП.
- Многостраничный каталог товаров.
- Фильтрация товаров по цене, типу, функционалу.
- Сортировка товаров по цене, типу, функционалу.
